## Brick definitions

### LEGO-System

There are 3 different definitions of a LEGO-System brick for the MagicaVoxelLibrary.

* LD (Low density) for 1 brick equals 1 voxel
* MD (Medium density) for 1 brick equals 2 \* 2 \* 3 (width \* depth \* height) voxel
